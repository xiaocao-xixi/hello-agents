# Extra10 - 两天入门学习计划（面试向）

> 目标：用两天时间尽量吃透 Hello-Agents 的核心内容，建立系统认知，并准备面试常见问答。

---

## 适用人群与学习目标

**适合人群**：第一次系统学习 Agent 的同学，具备基本 Python 与 LLM 概念基础。  
**时间投入**：2 天 × 8～10 小时（可根据自身情况增减）。

完成本计划后，你应该能够：
- 清晰讲出 Agent 核心组件、典型范式与多智能体协作方式
- 知道 Hello-Agents 教程的整体结构与核心章节脉络
- 能用项目代码复现 1～2 个关键范式或案例
- 构建一份面试“知识点 + 实例”清单

## 两天学习安排（面试向）

| 时间段 | 学习任务 | 输出要求 |
| --- | --- | --- |
| 上午 09:00-10:30 | 阅读第 1～2 章，梳理 Agent 定义、发展路线 | 3 句话解释“Agent 与 LLM 的区别” |

## 面试冲刺清单（建议最后 2 小时集中复盘）

**必会主题**
- Agent 的核心组件与工作流（思维、行动、观察）
- ReAct / Plan-and-Solve / Reflection 的区别
- Memory 系统（短期、长期、检索增强）
- Tool Use 的设计与安全性控制
- 多智能体协作的优势与挑战
- Agent 的评估维度与指标

**表达模板建议**
- “概念定义 → 项目章节对应 → 代码或案例举例”
- 重点关联 Hello-Agents 里的具体章节或案例，面试回答会更有“项目味”
